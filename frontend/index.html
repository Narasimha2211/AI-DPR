🏛️ AI DPR Analysis System

End-to-end AI platform automating evaluation of government DPRs with focus on MDoNER states. Combines NLP document intelligence, ML risk prediction, and explainable AI. Integrated with NeSETU for real-time project tracking.

📋 Key Features

  • NLP-based document parsing & section identification
  • Multilingual OCR — 10+ Indian languages
  • Quality scoring with 5 weighted criteria (A+ to F)
  • Compliance validation (Central + State + MDoNER)
  • ML risk prediction (XGBoost / LightGBM)
  • Monte Carlo simulation for cost quantification
  • Explainable AI with SHAP / LIME
  • Incremental learning — improves with each DPR
  • NeSETU dashboard integration
